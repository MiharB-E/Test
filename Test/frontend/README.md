# Frontend Setup

This is a Vite + React + TypeScript + Tailwind CSS application.

## Project Structure

- **/frontend**: Main directory for the frontend application.
- **/frontend/src**: Source code for the application.
- **/frontend/src/components**: Reusable components.
- **/frontend/src/pages**: Application pages.
- **/frontend/src/store**: Zustand store for state management.
- **/frontend/src/services**: API service calls with Axios.
- **/frontend/src/types**: TypeScript types for the application.

## Setup Instructions

1. Clone the repository.
2. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm run dev
   ```
